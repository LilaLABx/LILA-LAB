# LILA Lab — REST API

**Programmatic access to narrative indices and datasets.**

This directory will contain the LILA Lab REST API for querying narrative indices across languages and domains. The API is in the **planning stage** — this stub documents the intended design.

## Target Endpoints

```
GET  /v1/indices/{language}/{domain}
    → Monthly narrative index values for a language+domain pair
    → Query params: ?start=2020-01&end=2024-12&format=json|csv

GET  /v1/datasets/{language}
    → Dataset metadata and download links for a language

GET  /v1/languages
    → List all active pipeline languages with status

GET  /v1/aggregate
    → Cross-language aggregation
    → Query params: ?domain=economic&region=africa

GET  /v1/health
    → API health check
```

## Planned Stack

- Framework: FastAPI (Python)
- Authentication: API key (optional, for rate limiting)
- Documentation: OpenAPI/Swagger (auto-generated by FastAPI)
- Deployment: Docker container on VPS or serverless
