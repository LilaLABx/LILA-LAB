# Scripts

This folder is reserved for BENI v1.0 release scripts.

Recommended scripts to add:

- `build_beni_v1_articles.py`
- `resolve_reference_labels.py`
- `train_beni_v1_filter.py`
- `build_beni_v1_index.py`
- `validate_beni_v1_macro.py`

All scripts should read from and write to `BENI_v1_data_paper` paths, not older prototype paths.
