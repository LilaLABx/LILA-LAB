# YouTube Video Template

> Walkthrough, paper summary, or tutorial format.

---

## Video Types

### Type 1: Paper Summary (5 min)

| Section | Duration | Content |
|---------|----------|---------|
| Hook | 0:00–0:30 | "Here's a problem: [problem]. Here's what we did." |
| Background | 0:30–1:30 | Context — why this paper exists, what gap it fills |
| Method | 1:30–3:00 | The approach — pipeline diagram on screen |
| Results | 3:00–4:00 | Key findings — chart/table on screen |
| Impact | 4:00–4:30 | Why this matters — 2–3 implications |
| CTA | 4:30–5:00 | "Link in description. Subscribe for more." |

### Type 2: Pipeline Walkthrough (10–15 min)

| Section | Duration | Content |
|---------|----------|---------|
| Intro | 0:00–1:00 | What this pipeline does, what you'll learn |
| Setup | 1:00–3:00 | Clone repo, install dependencies |
| Data | 3:00–5:00 | Data format, how to prepare your own |
| Run | 5:00–10:00 | Run the pipeline step-by-step (screen recording) |
| Results | 10:00–12:00 | Interpreting output |
| Customize | 12:00–14:00 | How to adapt for your language |
| CTA | 14:00–15:00 | "Fork the repo. Submit an extension. Collab with us." |

### Type 3: Tutorial (10 min)

Same as walkthrough but more explanation per step, aimed at beginners.

## Thumbnail Guidelines

- Background: Use brand teal (`#006D77`) or light sand (`#FFDDD2`)
- Text: Bold white sans-serif, max 5 words
- Always include LILA Lab wordmark
- Include a relevant visual (pipeline diagram, chart, language script)
