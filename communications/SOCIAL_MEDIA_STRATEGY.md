# LILA Lab — Social Media Strategy

> How LILA Lab uses social media to build an audience of researchers, linguists, and contributors — and drive them into the collaboration funnel.

---

## Content Pillars

All social content comes from these 5 pillars. Every post should map to one.

| # | Pillar | Description | % of Content | Example |
|---|--------|-------------|-------------|---------|
| 1 | **Research output** | New papers, datasets, pipeline improvements, validation results | 40% | "Our BENI pipeline now achieves 91.7% accuracy. Here's what changed." |
| 2 | **Educational / How-to** | Tutorials, explanations of the pipeline, NLP concepts in low-resource context | 25% | "How to build an economic narrative index for YOUR language in 5 steps." |
| 3 | **Community & People** | New contributors, extension papers, annotation drives, contributor spotlights | 15% | "Welcome to our first AENI extension contributor — working on Assamese." |
| 4 | **Behind the research** | Methodological decisions, failures, lessons learned, interesting edge cases | 10% | "Why we switched from BERT to TF-IDF: a threading cost story." |
| 5 | **Calls to action** | Recruiting annotators, seeking language experts, call for replication, funding | 10% | "We need Assamese native speakers for a 2-hour annotation task." |

## Platform-Specific Strategies

### X / Twitter (`@LILA_Lab`)

| Element | Strategy |
|---------|----------|
| **Cadence** | 3–5 posts/week |
| **Format** | Short posts (1–2 sentences) + figure/visual. Threads (4–8 tweets) for deeper dives |
| **Best times** | Weekdays 9–11 AM Dhaka time, 8–10 AM ET (covers South Asia + US overlap) |
| **Engagement** | Reply to mentions within 24h. Quote-tweet relevant NLP papers. Participate in #NLProc, #ICLR, #ACL202X |
| **Thread structure** | Hook (1 tweet) → Method (2–3 tweets) → Results (1–2 tweets) → Call to action (1 tweet) |
| **Visuals** | Always attach a figure, chart, or pipeline diagram. Text-only posts underperform |

**Post templates:**

```
📢 New LILA Lab Technical Report: "[Title]"

We built [pipeline] that achieves [result] in [language].

Key finding: [one line result that surprises people]

🔗 Link: [GitHub/OSF/arXiv URL]
#LILALab #LowResourceNLP

[image: key figure from paper]
```

```
🧵 How we built [XENI pipeline] in [language]

1/N [Language] has [N] speakers but near-zero NLP infrastructure. We're changing that.

2/N The pipeline: [3-step diagram in text]
   → Collect news
   → LLM annotation
   → Classify + validate

3/N Key result: [impressive finding]

... [continue thread]

Final/N Want to do this for YOUR language?
   → COLLABORATION.md in our repo shows how.
   → Fork, extend, publish.
```

### LinkedIn

| Element | Strategy |
|---------|----------|
| **Cadence** | 2–3 posts/week |
| **Format** | Long-form posts (300–800 words), "story" structure. No threads needed |
| **Audience** | Academics, university departments, research institutions, policy organizations |
| **Structure** | Problem → What we did → What we found → Why it matters → Call to action |
| **Tags** | Tag relevant researchers, institutions, and organizations when applicable |

**Post structure:**

```
[Headline: The problem in one sentence — e.g., "84% of NLP is in English. Here's how we're fixing that for Bangla."]

[Paragraph 1: Why this matters — set up the problem with a human angle]
[Paragraph 2: What we built — the pipeline in plain language]
[Paragraph 3: What we found — one or two key results, no jargon]
[Paragraph 4: How YOU can contribute — the collaboration framework, link to repo]

[Call to action]
We're actively seeking linguistic experts for Assamese, Nepali, Sylheti, and more.
→ Link to COLLABORATION.md
#LILALab #NLP #LowResourceLanguages #OpenResearch

[Image: pipeline diagram or result chart]
```

### YouTube

| Element | Strategy |
|---------|----------|
| **Cadence** | 1 video/week |
| **Format** | 5–15 min walkthroughs. Screen recording + voiceover |
| **Series** | "LILA Lab Pipeline" series: each video walks through one pipeline component |
| **Audience** | Students, new contributors, visual learners |
| **SEO** | Title format: "LILA Lab: [What the video covers]" |
| **Description** | Always include: GitHub link, relevant DOI, COLLABORATION.md link, timestamps |
| **Thumbnail** | Text overlay summarizing the key takeaway |

**Video types:**

1. **Pipeline walkthrough** — "How BENI Classifies 664K Bangla News Articles" (12 min)
2. **Paper summary** — "Our Systematic Review of 20 Years of Economic Narrative Research in 5 Minutes" (5 min)
3. **Tutorial** — "Set Up the BENI Pipeline on Your Machine in 10 Minutes" (10 min)
4. **Contributor spotlight** — "How [Name] Built the AENI Assamese Pipeline" (8 min)
5. **Live stream** — Monthly lab updates + Q&A (30 min, once/month)

## Crisis & Coordination

| Situation | Action |
|-----------|--------|
| Negative comment / criticism | Reply constructively within 24h. Acknowledge, address, improve. |
| Correction needed (error in published post) | Post correction pinned to thread. Update original if possible. |
| Harassment | Do not engage. Block and report. Document incident. |
| Co-author disagreement on results | Do not post. Resolve offline. Retract post if published. |
| Platform outage | Announce on other channels. Resume normal cadence when restored. |
