# Paper 2: BENI v1 — Bangla Economic Narrative Dataset

**Full Title:** BENI v1: A Dataset of 933,000 Bangla News Articles Annotated for Economic Narrative Analysis

**Status:** Complete (OSF preprint + Mendeley data release)

---

This paper describes the release of the BENI v1 dataset: the Potrika corpus (664K raw articles) plus LLM-annotated labels (economic relevance, topic, sentiment, frame) and a processed monthly index. The dataset is the empirical foundation for Papers 1 and 3.

The actual paper lives at:

`dataset/BENI/beni-v1/osf_upload_package/paper/main.tex`

### Key outputs

- **Corpus**: 933,000 Bangla news articles from 7 newspapers (2014–2024)
- **Annotation**: LLM ensemble (Claude + GPT-4o) — 12-field economic narrative schema
- **Data**: Zenodo, Mendeley Data, OSF (CC BY 4.0)
